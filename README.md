# AI_Classificazione_CorpiCelesti

Questo progetto applica tecniche di machine learning per classificare oggetti astronomici in tre categorie:
- Galassie
- Stelle
- Quasar

L'analisi è basata su dati fotometrici (indici ugriz) e sulle loro trasformazioni (differenze di colore), con l'obiettivo di confrontare le prestazioni di diversi algoritmi di classificazione.

Obiettivi:
- Analizzare dati astronomici
- Simulare fenomeni astrofisici
- Visualizzare risultati in modo chiaro
- Applicare metodi scientifici e computazionali

Linguaggi utilizzati: Python, Jupyter Notebook, NumPy, Matplotlib.

I dati sono stati analizzati grazie a diversi algoritmi di ML:
- K-Nearest Neighbour;
- Naive Beyes;
- Random forest.
